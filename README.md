## 故事簡介

故事的背景設定於月亮計畫世界觀中都市的一間能源公司「腦葉公司」。
腦葉公司收容大量「異想體」，藉由對這些怪物們工作來產出能源。作為一名資深員工，鎮壓出逃的異想體，解救受困設施深處的同伴吧！

---

## 核心玩法

* **2D 平台 Boss Rush**：專注於高強度的 Boss 戰鬥體驗。
* **觀察與鎮壓**：仔細觀察，閃躲敵人的攻擊，運用多樣武器進行鎮壓。
* **控制器支援**：支援 **鍵盤** 和 **XBox 手把** 遊玩，提供流暢的操作手感。
